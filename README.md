# Featherstill Capstone Project

## Overview

This repository contains the source code and documentation for the Capstone Project developed for **SER 517: Software Factory I** as part of the **Master of Science in Software Engineering** program at **Arizona State University (ASU)**.

## Industry Sponsor

This project is proudly sponsored by **Featherstill**, providing industry context and requirements for our solution.

## Project Structure

The codebase is organized into three main components:

*   **`frontend/`**: Contains the user interface and client-side application logic.
*   **`backend/`**: Hosts the server-side application, API definitions, and database integration.
*   **`hardware/`**: Includes firmware, schematics, and hardware-related documentation.
